Details

Data collected using Google Analytics from September 1st, 2016 to February 26th, 2017. Each "facet use" consists of a catalog user selecting a link under Koha's "refine your search" sidebar. Searches with preset facets (e.g. a search from the Materials Library home page) would not count.

The "x (remove)" entry under some categories occurs when a user removes any facet from their search refinements. Similarly, "Show more" is not the value of a facet but indicates when a user wanted to see more options under the category.

Uses a d3 treemap zoomable template.
